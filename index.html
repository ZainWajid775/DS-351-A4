Tree & Graph
Visualizations

Animal Kingdom · Global Stock Markets

Tree Dataset

Animal Kingdom

Source: UCI Zoo Classification dataset.
Nodes: Root (Animal Kingdom), 7 class nodes, 101 animal species.
Edges: Parent → child biological classification (Kingdom → Class → Species).

Graph Dataset

Global Stocks Traded

Source: World Bank – Stocks traded, total value (USD).
Nodes: Countries / economic regions.
Edges: Countries with similar-scale trading volumes (within 5× ratio) are connected.

Part 1

Tree Visualizations

Layout 1

Hierarchical (Top-Down)

Classic taxonomy tree. Each level — Kingdom, Class, Species — is laid out on its own horizontal tier, making the 3-level hierarchy immediately readable.

Layout 2

Radial / Spring Layout

Classes radiate outward from the central root node. Provides an organic, exploratory view of how animal classes branch from a common origin.

Part 2

Graph Visualizations

Layout 1

Spring / Force-Directed

Repulsion–attraction forces push similar-volume countries together. High-volume markets (USA, China) naturally cluster near the centre.

Layout 2

Circular Layout

Countries arranged in a ring, sorted by continent. Edges crossing the circle reveal cross-regional trading-volume similarities at a glance.

Layout 3 · Bonus

Kamada-Kawai

Energy-minimisation algorithm places similar-volume countries spatially close. Reinforces continental clustering while minimising edge crossings.
